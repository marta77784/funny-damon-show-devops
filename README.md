# YouTube Stats Tracker for Funny Damon Show

Python-скрипт для автоматического сбора статистики канала @funnydamon через YouTube Data API v3.

## Возможности
- Получает количество подписчиков, просмотров, видео
- Сохраняет данные в JSON с временной меткой
- Готов к автоматизации (GitHub Actions)

## Технологии
- Python (google-api-python-client)
- Git + GitHub

## Запуск
1. Установи зависимости: `pip install google-api-python-client`
2. Вставь свой API-ключ в скрипт
3. Запусти: `python youtube_stats.py`
